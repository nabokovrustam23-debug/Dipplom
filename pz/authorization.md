2.1 Авторизация пользователей

При обращении к CRM-системе «Тихий час» неавторизованному пользователю доступна только главная страница с информацией о филиалах, услугах и мастерах, а также форма отправки заявки. Для доступа к функционалу онлайн-записи и управления требуется авторизация.

2.1.1 Страница входа

Страница входа должна содержать следующие элементы управления:

– текстовое поле для ввода логина;
– текстовое поле для ввода пароля;
– кнопку «Войти».

При вводе неверных учётных данных система должна отображать сообщение об ошибке. После успешной авторизации пользователь перенаправляется на страницу, соответствующую его роли: клиент – в личный кабинет с записами, мастер – на расписание, администратор – в журнал записей, владелец – на панель управления.

2.1.2 Регистрация клиентов

Страница регистрации должна содержать следующие элементы управления:

– текстовые поля для ввода фамилии, имени и отчества;
– текстовое поле для ввода номера телефона;
– текстовое поле для ввода адреса электронной почты;
– текстовое поле для ввода пароля;
– кнопку «Зарегистрироваться».

После успешной регистрации система автоматически авторизует пользователя и перенаправляет его в личный кабинет.
