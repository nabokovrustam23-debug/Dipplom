2.2 Руководство по стилю

Все экранные формы пользовательского интерфейса CRM-системы «Тихий час» должны иметь заголовок, отражающий назначение текущей страницы или диалогового окна. На главной странице приложения и в подвале сайта должен быть установлен логотип, изображённый на рисунке Б.1. Логотип запрещается искажать: не допускается изменение изображения, его пропорций или цвета. Для приложения должна быть установлена иконка, изображённая на рисунке Б.1.

В качестве основного шрифта интерфейса используется гарнитура Inter. Для заголовков и бренд-элементов применяется акцидентный шрифт Cormorant Garamond. Размер основного текста — 14 пикселей, межстрочный интервал — 1,6. Моноширинный шрифт (ui-monospace, Cascadia Code, Menlo) используется для отображения технических данных.

Цветовая палитра «Латунь на тёплом угле» включает следующие основные цвета. Основной фон страниц (1F1B17) представляет собой тёмный тёплый оттенок, используемый для всех окон и страниц приложения. Фон карточек филиалов, услуг, бронирований и модальных окон выполняется цветом 2A251F. Для полей ввода и бейджей применяется более тёмный оттенок 332C24. Цвет границ элементов — 3B342C.

Основной текст выполняется цветом E8E0D2, вспомогательные надписи и подсказки — цветом A39685. Акцентный золотой цвет B89968 применяется для кнопок записи, подтверждения и оплаты, а также для ссылок и активных интерактивных элементов. Текст на акцентных кнопках выполняется тёмным цветом 1F1B17. При наведении курсора используется более светлый оттенок CCAE7E.

Для сообщений об ошибках и опасных действий используется цвет A8675F, а для подтверждений успешных операций — 7A8471.

Скругления углов составляют 12 пикселей для карточек и модальных окон, 8 пикселей для кнопок и полей ввода. Бейджи и тэги имеют полностью скруглённую форму (999 пикселей). Для теней используются значения: малые элементы — 0 1px 2px rgba(0, 0, 0, 0.20), модальные окна и крупные панели — 0 4px 16px rgba(0, 0, 0, 0.30).

Максимальная ширина контента составляет 1200 пикселей. Адаптивность реализована для двух точек перехода: планшеты (960 пикселей) и мобильные устройства (720 пикселей).
